## Running locally

1. Install Maestro: `curl -Ls "https://get.maestro.mobile.dev" | bash`
2. Boot an iOS simulator matching the target device size.
3. Build and install Surve on the simulator.
4. Run:

```bash
# All devices, all flows
npm run screenshots

# Single device (6.7-inch, 6.5-inch, or 5.5-inch)
npm run screenshots:6.7
npm run screenshots:6.5
npm run screenshots:5.5

# Single flow (use --flow= with tsx directly)
npx tsx src/scripts/generate-screenshots.ts --flow=listing_detail

# Both filters
npx tsx src/scripts/generate-screenshots.ts --device=6.7 --flow=home
```

Screenshots land in `docs/screenshots/<device-size>/<flow-label>.png`.

Switch between simulators and re-run with `--device=<size>` to capture each frame set.
