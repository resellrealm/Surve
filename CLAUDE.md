# Point! — iOS Survey App

## Project Info
- React Native + Expo + TypeScript
- Supabase backend (project: pfardxvzeatcaezfiqah)
- Design system: Grey/bluish-grey theme #475569, iOS-native typography
- Routes in src/app/, components in src/components/, lib in src/lib/

## Commands
- Type check: `npx tsc --noEmit`
- Start dev: `npx expo start`

## Key Files
- Theme: src/constants/theme.ts
- Supabase: src/lib/supabase.ts
- Store: src/lib/store.ts
- Types: src/types/index.ts
